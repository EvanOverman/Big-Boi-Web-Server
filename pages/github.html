Auto-Web-Server

Auto-Web-Server is a project aimed to make developing a website just a little bit easier, by automatically programming all of the Node.js the program will allow you to work quickly, directly on a local machine as though it were a server, make quick updates, and get you site up faster. This website currently uses Auto-Web-Server.
